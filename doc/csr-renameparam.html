Software::JRefactory - Rename Parameter

Rename Parameter

Using the UML diagram, right click on the method where the parameter resides. Select the method refactoring menu, followed by the rename parameter menu. Select the parameter you wish to rename.

A dialog box appears where you enter the new name of the parameter, press OK and the work is done.

This is the first refactoring on the path to create a rename method refactoring.
