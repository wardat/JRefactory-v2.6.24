Software::JRefactory - Metrics

Metrics are also available. To generate a metrics window, simply right click on the UML class diagram. You have four options.

The first option is to get metrics on the entire project. Essentially, the project includes all packages that have been loaded and can be selected in the Package Selection window.

The second option is to get metrics on the current package.

The third option is to get metrics on the current class.

If you right clicked on a specific method, then a fourth option is available to get information about a specific metric.

Additional metrics will be added as they become available.

For a list of the metrics that are available, see the description of the command line metrics tool.
